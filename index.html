소시오톡 (SocioTalk)

GitHub Pages + Apps Script + Gemini API 대담 시스템
💡 사용 방법: 대담하고 싶은 현대 사회의 사상가를 먼저 선택하세요.
그 후 하단 입력창에 현대 사회의 쟁점이나 궁금한 점을 질문하면 사상가의 관점으로 답변을 받게 됩니다.
대담할 사상가 선택
사상가를 선택하시면 대담 준비 상태가 됩니다.
질문 기입란
사상가의 답변
사상가를 선택하고 질문을 던지면 이곳에 심오한 철학적 통찰이 기록됩니다.
단축키: 입력창에서 Ctrl + Enter를 누르면 대담 질문이 즉시 전송됩니다.
답변이 넘어오지 않을 경우 구글 웹 앱 배포 시 권한이 [Anyone](모든 사람)으로 설정되었는지 재확인하세요.
